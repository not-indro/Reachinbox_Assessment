# ReachInBox Frontend Assignment

## Overview
This project is a frontend application for ReachInBox, developed using Next.js and TypeScript as part of an assignment. The application provides essential email management features with a modern and responsive interface.

## Technologies Used
- **TypeScript**: Enhances code reliability with type safety.
- **Next.js**: A powerful React framework for server-side rendering and static site generation.
- **Tailwind CSS**: Utility-first CSS framework for efficient styling.
- **Axios**: Promise-based HTTP client for API requests.
- **jwt-decode**: Library for decoding JSON Web Tokens.

## Deployment
The application is deployed on Vercel and can be accessed [here](https://reachinbox-assesment.vercel.app/).

## Demo Video
Watch a demonstration of the application [here](https://www.loom.com/share/f3d1c2725b8a4b1896fc3f3760eeb9dc).

## How to Run Locally

### Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/bhuvanesh-hj/reachinbox_assesment.git
   ```
2. **Navigate to the project directory**:
   ```bash
   cd reachinbox_assesment
   ```
3. **Install the dependencies**:
   ```bash
   npm install
   ```
4. **Start the development server**:
   ```bash
   npm run dev
   ```
5. **Open your browser and visit**:
   ```bash
   http://localhost:3000
   ```

## Features
- **Authentication**: Secure user login and token management.
- **Fetch Emails**: Retrieve emails from the server.
- **Send Emails**: Compose and send new emails.
- **Delete Emails**: Remove unwanted emails.
